# Reset the gretaR Global Environment

Clear all tracked nodes, distributions, and reset the node counter.
Called automatically on package load; call manually to start a fresh
model definition.

## Usage

``` r
reset_gretaR_env()
```

## Value

Invisible `NULL`.

## Examples

``` r
if (FALSE) { # \dontrun{
reset_gretaR_env()
} # }
```
