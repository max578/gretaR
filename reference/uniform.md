# Uniform Distribution

Create a uniformly-distributed variable on the interval
`[lower, upper]`.

## Usage

``` r
uniform(lower = 0, upper = 1, dim = NULL)
```

## Arguments

- lower:

  Lower bound (numeric).

- upper:

  Upper bound (numeric).

- dim:

  Integer vector of dimensions (default scalar).

## Value

A `gretaR_array`.

## Examples

``` r
if (FALSE) { # \dontrun{
p <- uniform(0, 1)
} # }
```
